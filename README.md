## setup
1. install nodejs
2. go into directory 'travelflan-app'
3. npm install

## gcm config
1. modify Config.js
2. GCM Registration Token can be viewed through the android studio Log

## start server
1. npm start
2. http://localhost:3000

## send message
1. click send message button, you should get a message on android device

## improvement
